Overview

This lab guides you through setting up your DSPy development environment. A properly configured environment is essential for building and testing DSPy programs effectively.

Learning Objectives

  • Install DSPy and required dependencies
  • Configure LLM provider credentials
  • Set up Python development environment
  • Validate installation with test programs
  • Understand different LLM provider options

Prerequisites

  • Python 3.8 or higher
  • pip (Python package manager)
  • Basic command line familiarity
  • API key for at least one LLM provider

Installation Steps

1. Check Python Version

python --version
# Should show Python 3.8 or higher

2. Create Virtual Environment (Recommended)

# Create virtual environment
python -m venv dspy_env

# Activate virtual environment
# On Linux/Mac:
source dspy_env/bin/activate
# On Windows:
dspy_env\Scripts\activate

3. Install Dependencies

pip install -r requirements.txt

This will install DSPy and required packages for LLM integration.

4. Configure LLM Provider

Set your API key as an environment variable:

# For OpenAI
export OPENAI_API_KEY="your-api-key"

# For Anthropic
export ANTHROPIC_API_KEY="your-api-key"

# For other providers, check their documentation

5. Validate Installation

import dspy
print(f"DSPy version: {dspy.__version__}")

LLM Provider Options

OpenAI

  • GPT-3.5, GPT-4 models
  • High-quality responses
  • Requires API key

Anthropic

  • Claude models
  • Strong reasoning capabilities
  • Requires API key

Local Models

  • Use with Hugging Face transformers
  • No API key required
  • Requires GPU for best performance

Test Program

Create a simple test to verify your setup:

import dspy

# Configure your LLM provider
import os
api_key = os.getenv("OPENAI_API_KEY")

# Create a simple signature
class SimpleQA(dspy.Signature):
    question = dspy.InputField()
    answer = dspy.OutputField()

# Test the setup
qa = dspy.Predict(SimpleQA)
result = qa(question="What is 2 + 2?")
print(f"Result: {result.answer}")

Troubleshooting

Import Errors

Problem: ModuleNotFoundError: No module named 'dspy'

Solution: pip install dspy-ai

API Key Errors

Problem: API authentication fails

Solution: Verify your API key is set correctly and has sufficient credits

Version Conflicts

Problem: Dependency conflicts

Solution: pip install --upgrade pip && pip install --upgrade dspy-ai
