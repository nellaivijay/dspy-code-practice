DSPy Code Practice

Free Hands-on Labs for Declarative LLM Programming

🎯 Educational Mission

A comprehensive, vendor-independent DSPy learning environment designed for developers, data engineers, and AI practitioners who want to master declarative LLM programming through hands-on practice.

🎓 Learning Approach

Our labs are designed to build knowledge progressively:

  • Foundation (Labs 0-2): Setup, fundamentals, and first programs
  • Core Concepts (Labs 3-5): Signatures, modules, evaluation, and optimization
  • Advanced Patterns (Labs 6-8): Complex reasoning, pipelines, and RAG
  • Production Applications (Labs 9-11): Agents, real-world apps, and deployment

🚀 Quick Start

Get started with DSPy learning:

  1. Clone the repository: git clone https://github.com/nellaivijay/dspy-code-practice.git
  2. Install dependencies: pip install -r requirements.txt
  3. Set up your LLM API key
  4. Start with Lab 0: Environment Setup
